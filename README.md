# Website Traffic Dashboard

## Overview
A live interactive dashboard tracking website sessions, users, bounce rate,
and traffic sources built with Looker Studio and Google Analytics data.

## Live Dashboard
Click here to view: [https://lookerstudio.google.com/s/u7chVy9P-YE]

## Tools Used
- Looker Studio
- Google Analytics (GA4 Demo Account)
- Google Sheets

## Key Metrics Tracked
- Total Sessions
- Active Users
- Bounce Rate
- Traffic by Source / Medium
- Users by Country
- Top Pages by Views

## Dataset
Google Analytics GA4 Demo Account (Google Merchandise Store)
